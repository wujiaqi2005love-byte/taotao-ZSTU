# 车辆动力学分析与弹簧选型整合平台

## 软件简介
本软件为车辆动力学分析与弹簧选型的集成平台，支持ISO 8608路面激励、ISO 2631舒适性评价、七自由度动力学模型仿真、刚度搜索与弹簧选型等功能。界面简洁，操作直观，适合科研、教学及工程应用。

## 主要功能
- **舒适度分析**：基于ISO 2631标准，对四座位加速度进行舒适性评价。
- **四轮统一刚度搜索**：通过频率约束和RMS扫描，自动搜索最优统一刚度，并可一键发送到弹簧选型。
- **前后分离刚度搜索**：支持前后刚度分离优化，二维参数空间搜索，满足前后比例约束。
- **弹簧选型系统**：自动接收最优刚度，输出弹簧规格参数与3D预览，支持自定义预压缩量。

## 软件结构
```
main.py                  # 主程序入口，集成主菜单和窗口调度
core/
    models.py            # 动力学核心模型与参数定义
threads/
    main_thread_runner.py
    simulation_threads.py# 仿真多线程实现
utils/
    calculators.py       # 计算工具函数
    shared_state.py      # 全局状态管理
    window_base.py       # 窗口基础类
windows/
    comfort_analysis.py  # 舒适度分析窗口
    uniform_stiffness.py # 统一刚度搜索窗口
    separate_stiffness.py# 分离刚度搜索窗口
    spring_selector.py   # 弹簧选型窗口
```

## 使用方法
1. **运行主程序**：
   - Windows下双击`main.py`或在命令行输入：
     ```bash
     python main.py
     ```
2. **主界面说明**：
   - 左侧为功能菜单，依次为舒适度分析、统一刚度搜索、分离刚度搜索、弹簧选型。
   - 建议先进行刚度搜索，获得最优刚度后自动进入弹簧选型。
3. **弹簧预压缩量设置**：
   - 在弹簧选型窗口下方，可分别填写四个方位（左前、右前、左后、右后）弹簧的预压缩量（单位mm），用于更精确的弹簧选型。
4. **数据流说明**：
   - 刚度搜索窗口计算结果可一键发送到弹簧选型系统，实现参数自动传递。
5. **结果导出**：
   - 各窗口均支持结果导出与保存，便于后续分析与汇报。
